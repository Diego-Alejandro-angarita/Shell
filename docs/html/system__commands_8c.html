Comando CALC (Calculadora)

Realiza operaciones matemáticas simples.

Ejecuta operaciones aritméticas básicas.

Soporta suma (+), resta (-), multiplicación (* o x) y división (/). Realiza validación de argumentos y manejo de errores (incluyendo división por cero).
