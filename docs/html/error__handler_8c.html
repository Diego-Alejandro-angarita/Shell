Detailed Description

Manejo de errores en la shell y utilidades visuales.

Este archivo centraliza el manejo de errores y funciones visuales de la shell EAFITos.

Proporciona:

  • Impresión de errores en stderr con formato ANSI.
  • Limpieza de pantalla.
  • Banner de bienvenida estilizado.
