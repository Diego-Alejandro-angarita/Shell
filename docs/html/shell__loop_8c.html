Detailed Description

Implementación del bucle principal y despacho de comandos.

Este módulo implementa el REPL (Read-Eval-Print Loop) de la shell EAFITos.

Funcionalidades principales:

  • Registro dinámico de comandos.
  • Búsqueda mediante tabla de punteros a función.
  • Ejecución y manejo de errores.

◆ loop_shell()

void loop_shell ( )

Bucle principal Read-Eval-Print Loop (REPL).

Inicia el bucle principal de la shell. No retorna hasta que el comando 'salir' sea invocado.

Ciclo de vida:

  1. Read: Muestra el prompt "EAFITos>" y lee la entrada.
  2. Parse (Eval implícito): Divide la entrada en argumentos.
  3. Execute: Busca y corre el comando.
  4. Loop: Repite o libera memoria.
